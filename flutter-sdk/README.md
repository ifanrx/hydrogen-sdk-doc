# 知晓云 Flutter SDK

## 简介

知晓云 Flutter SDK 是方便开发者结合知晓云产品，进行 Flutter 应用开发的工具包，通过该 SDK，你可以在 Flutter 中操作存储在知晓云中的数据表、内容库、媒体文件，以及调用云函数进行后端逻辑的执行。

详细内容，可查看以下功能模块介绍：

* [用户](./user/account.md)
* [数据表](./schema/README.md)
